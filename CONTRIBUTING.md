# Contributing to Carceris

Carceris is licensed under AGPLv3.

## Development Standards

- Keep changes small and reviewable.
- Preserve MAJOR.MINOR.PATCH versioning.
- Do not commit real facility data.
- Do not commit `config.local.php`.
- Run PHP syntax checks before release.
- Update release notes for every release.
- Add migrations for database changes.
- Keep security in mind for every route and form.

## Pull Requests

A pull request should include:

```text
summary of changes
testing performed
security considerations
migration notes if applicable
documentation updates if applicable
```
