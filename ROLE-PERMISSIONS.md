# Role Permissions

## Admin

Admins can:

```text
create log entries
view archives
print logs
download logs
send completed logs
correct and void entries
manage users
manage categories
manage settings
manage Log Delivery
run upgrades
view Audit
export Audit CSV
prune Audit records
view System Status
```

## Supervisor

Supervisors can:

```text
create log entries
view archives
print logs
download logs
send completed logs when Log Delivery is configured
correct and void entries
view and search Audit
review officer activity
```

Supervisors cannot:

```text
manage users
manage settings
manage Log Delivery settings
run upgrades
export Audit CSV
prune Audit records
```

## Officer

Officers can:

```text
create log entries
view archives
print logs
download logs
```

## Viewer

Viewers can:

```text
view archives
print logs
download logs
```
