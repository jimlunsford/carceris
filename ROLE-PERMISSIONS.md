# Role Permissions

## Admin

Admins can:

```text
view the live Active Log
create log entries
view archives
print logs
use Daily Logs downloads and manual sends
correct and void entries
manage users
manage categories
manage settings
manage Log Delivery
run backups and restores
run upgrades
view Audit
export Audit CSV
prune Audit records
view System Status
```

## Supervisor

Supervisors can:

```text
view the live Active Log
create log entries
view archives
print logs
use Daily Logs downloads
send completed logs when Log Delivery is configured
correct and void entries
view and search Audit
review officer activity
```

Supervisors cannot:

```text
manage users
manage settings
manage Log Delivery settings
run backups or restores
run upgrades
export Audit CSV
prune Audit records
view System Status
```

## Officer

Officers can:

```text
view the live Active Log
create log entries
view archives
print logs
```

## Viewer

Viewers can:

```text
view archives
print dated logs from the Archive
```

Viewers cannot:

```text
view the live Active Log
create log entries
open the live Print view
use Daily Logs downloads or sends
```
