# Admin Guide

## First Tasks

After installation:

1. Confirm System Status.
2. Set Environment Mode.
3. Configure Header Branding.
4. Configure Login Branding.
5. Configure Print View Branding.
6. Configure Daily Log filename pattern.
7. Configure Operational Day start time.
8. Create supervisor and officer users.
9. Configure Log Delivery if email delivery will be used.
10. Test backup and restore.

## Daily Logs

Daily Logs is the operations dashboard for completed logs.

Use it to:

```text
review recent completed logs
download PDF, text, or HTML
send completed logs
review failed sends
resend failed deliveries
review scheduled delivery history
```

## Archive

Archive is for deeper lookup and review.

Use it to:

```text
search historical logs
open a specific log
review entries
print a log
download PDF
access correction and void history
```

## Audit

Audit is for accountability.

Use it to:

```text
review logins
review user actions
review corrections
review voids
review delivery actions
review admin actions
```

Supervisors can view/search Audit. Audit pruning and export remain Admin-only.


## Backup & Restore

Use Admin -> Backup & Restore to download a protected backup bundle.

Only admins can access this area.

Restore is destructive and should only be used after an external backup has been created and facility leadership has approved the action.
